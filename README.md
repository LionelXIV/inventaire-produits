# Mini Inventaire de Produits

Exercice JavaScript - Séance 08 - INF37407

## Description
Application web pour gérer un inventaire de produits avec calcul automatique de la valeur totale.

## Fonctionnalités
- Ajout de produits (nom, prix, quantité)
- Affichage dans un tableau dynamique
- Calcul automatique de la valeur totale
- Formatage des prix en dollars canadiens
- Interface responsive avec Bootstrap 5

## Technologies utilisées
- HTML5
- CSS (Bootstrap 5)
- JavaScript (ES6)
- Programmation Orientée Objet (POO)

## Structure du projet
- `index.html` - Structure de la page web
- `script.js` - Logique JavaScript (classes Produit et Inventaire)
- `style.css` - Styles CSS (optionnel, Bootstrap utilisé)
- `README.md` - Documentation du projet

## Concepts JavaScript utilisés
- Classes ES6
- Constructeurs et méthodes
- Manipulation du DOM
- Gestion d'événements
- Tableaux et boucles
- Formatage de nombres

## Utilisation
1. Ouvrir `index.html` dans un navigateur web
2. Remplir le formulaire avec les informations du produit
3. Cliquer sur "Ajouter le produit"
4. Le produit apparaît dans le tableau avec calcul automatique
